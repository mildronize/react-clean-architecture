## Running

### Development

1. Install dependencies

    ```shell script
    $ yarn
    ```

1. Start the web-app next.js application

    ```shell script
    $ yarn workspace @project/web-app dev
    ```
   
   Then the application should be accessible on [http://localhost:3000](http://localhost:3000).
